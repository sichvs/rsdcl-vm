# rsdcl-vm
*Rsd e um projeto que foi criado para testar projetos, compilar arquivos etc, em uma maquina virtual, ou um terminal virtual (windows, ubuntu, linux)*

# Api
*Com a rsd-api você consegue se comunicar com seu celular/computador, com notificações, vibrações de coisas que acontecem na maquina.*

# Instalação da Api
*Para isto, você deve instalar o repositório da api, depois colocar o caminho dele no arquivo* `$PREFIX/var/.rsd-cl/hdd.properties`

# Comandos
Use `rsd` para listar os comandos disponíveis no rsdcl-vm.

# Instalação (linux/termux)
Primeiro, clone o repositório no $HOME do seu terminal.
`cd $HOME`
`git clone https://github.com/sichvs/rsdcl-vm`

*Depois, iremos entrar no repositório e executar o binario.*
`cd rsdcl-vm`
`chmod a+x execme`
`./execme`

*Feito! Você tem uma maquina virtual/terminal, em seu termux/linux!*
